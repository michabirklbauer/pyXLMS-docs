# Supported pyXLMS I/O

The following section gives an overview of the supported crosslink search engines
and input file formats for reading crosslink results, as well as the supported export options.

The following abbreviations are used:
- **CSMs:** crosslink-spectrum-matches
- **XLs:** crosslinks/peptide pairs/residue pairs

## Supported Search Engines and Formats

The following crosslink search engines and formats are currently supported:

| Search Engine/Format | Parser            | CSMs | XLs | Supported Formats                           | API Docs |
|:---------------------|:------------------|:----:|:---:|:--------------------------------------------|:---------|
| MaxQuant/MaxLynx     | `read_maxquant()` | ✅  | ❌  | `.txt`, any text-like                       | [link](https://hgb-bin-proteomics.github.io/pyXLMS/pyXLMS.parser.html#pyXLMS.parser.read_maxquant) |
| MeroX                | `read_merox()`    | ✅  | ❌  | `.csv`, `.zhrm`, any text-like              | [link](https://hgb-bin-proteomics.github.io/pyXLMS/pyXLMS.parser.html#pyXLMS.parser.read_merox) |
| MS Annika            | `read_msannika()` | ✅  | ✅  | `.pdResult`, `.xlsx`, `.txt`, any text-like | [link](https://hgb-bin-proteomics.github.io/pyXLMS/pyXLMS.parser.html#pyXLMS.parser.read_msannika) |
| pLink 2 & 3          | `read_plink()`    | ✅  | ✅  | `.csv`, any text-like                       | [link](https://hgb-bin-proteomics.github.io/pyXLMS/pyXLMS.parser.html#pyXLMS.parser.read_plink) |
| Scout                | `read_scout()`    | ✅  | ✅  | `.csv`, any text-like                       | [link](https://hgb-bin-proteomics.github.io/pyXLMS/pyXLMS.parser.html#pyXLMS.parser.read_scout) |
| xiSearch & xiFDR     | `read_xi()`       | ✅  | ✅  | `.csv`, any text-like                       | [link](https://hgb-bin-proteomics.github.io/pyXLMS/pyXLMS.parser.html#pyXLMS.parser.read_xi) |
| xiNET & xiVIEW       | `read_xinet()`    | ✅  | ✅  | `.csv`, any text-like                       | [link](https://hgb-bin-proteomics.github.io/pyXLMS/pyXLMS.parser.html#pyXLMS.parser.read_xinet) |
| XlinkX               | `read_xlinkx()`   | ✅  | ✅  | `.pdResult`, `.xlsx`, `.txt`, any text-like | [link](https://hgb-bin-proteomics.github.io/pyXLMS/pyXLMS.parser.html#pyXLMS.parser.read_xlinkx) |
| mzIdentML            | `read_mzid()`     | ✅  | ❌  | `.mzid`                                     | [link](https://hgb-bin-proteomics.github.io/pyXLMS/pyXLMS.parser.html#pyXLMS.parser.read_mzid) |
| pyXLMS/custom        | `read_custom()`   | ✅  | ✅  | `.xlsx`, `.parquet`, `.csv`, any text-like  | [link](https://hgb-bin-proteomics.github.io/pyXLMS/pyXLMS.parser.html#pyXLMS.parser.read_custom) |

_\*any text-like denotes all `.csv` like file formats that use a character delimiter e.g. `.csv`, `.tsv` and tab-delimited `.txt`._
